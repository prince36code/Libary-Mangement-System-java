# Library Management System
 Project Overview
The **Library Management System** is a simple Java console-based application created to understand and practice **core Object-Oriented Programming (OOP) concepts** and to learn **Git and GitHub workflows**.

This project is part of my learning journey as a Computer Science student.

---

 Project Objectives
- Learn and apply core OOP principles in Java
- Understand Java packages and imports
- Practice clean project structure
- Learn Git and GitHub (clone, commit, push)
- Build confidence in writing and organising real projects

---

 OOP Concepts Used
- **Classes and Objects**
- **Encapsulation** (private fields with public methods)
- **Constructors**
- **Packages**
- **Abstraction** (basic separation of concerns)

---

 Features
- Add books to the library
- Manage student information
- Display book and student details
- Organised package-based structure
 More features will be added as learning progresses.
